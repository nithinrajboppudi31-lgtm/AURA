# AURA - Luxury Minimalist E-Commerce Web Template

AURA is a high-converting, fully responsive E-Commerce web layout built with HTML5, Tailwind CSS, and Lucide Icons.

## File Structure
- `index.html`: Main landing page, collection grid, and sliding cart drawer.
- `product.html`: Single product detail layout with variant selections.
- `checkout.html`: Streamlined multi-step checkout screen layout.

## Setup Instructions
1. Upload these files to any web host, or drop the directory straight into Vercel or Netlify.
2. Edit product names and image links directly inside the `.html` files.
